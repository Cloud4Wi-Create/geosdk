# ComparisonCondition

Data Model used to indicate a condition for any Metric of any Resource or any aggregated Metric of any Statistic.

Name        |Type      
------------|----------
eq | `Integer` 
neq | `Integer` 
lt | `Integer` 
lte | `Integer` 
gt | `Integer` 
gte | `Integer` 


```json
{
    "gt": 10,
    "lte": 20
}
```
