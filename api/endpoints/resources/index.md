# Resource Endpoints

This section documents a the endpoints provided to perform CRUD operations on [Resources](/api/concepts/resource-definition.md).

More in detail, the following main categories of Resource Endpoints are defined.

* **[Platform-created Resource Endpoints](/api/endpoints/resources/platform-created/index.md)**. Allow to perform the READ operation on [Platform-related Resources](/api/resources/platform-created/index.md).
* **[User-created Resource Endpoints](/api/endpoints/resources/user-created/index.md)**. Allow to perform all the CRUD operations on [User-created Resources](/api/resources/user-created/index.md).
